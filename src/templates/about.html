I created this site to make the process of making up release notes much simpler. I found that I was copy and pasting a table from Jira into a Google Document. After I did that I would go through and remove some columns and adjust some of the formatting and fonts. Once I was done all of that, I would go through all of the issues and make up a list of "TL;DR" items, which is kind of like a summary.

This site allows me to copy the same table I already was, but now I have someting that does all the formatting for me! And all of you, too.
